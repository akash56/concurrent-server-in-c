The program is done in C language
THe files located on the multiserv zip are client.c server.c and three text files with random text of 20 words.
Running the program:
1. First compile the server.c program with command "gcc -o server server.c"
2. Run server with command "./server"
3. Now compile the client.c program with command "gcc -o client client.c"
4. Run client with command "./client [server IP address] [Text file indicatorin this case 1 or 2 or 3]"
Example: ./client 192.168.0.22 2
